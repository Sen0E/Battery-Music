# kugou_music_api_dart 🎵

一个酷狗音乐 API 的 Dart 版本实现，基于开源项目 [KuGouMusicApi](https://github.com/MakcRe/KuGouMusicApi)。

🎵 访问酷狗音乐的各种功能  
🎧 搜索歌曲、专辑、歌手等音乐资源  
📱 专为 Dart 和 Flutter 项目打造  

## 特性 ✨

- 🔍 歌曲搜索功能
- 🎶 获取歌曲详情
- 👤 歌手信息查询
- 💾 专辑信息获取
- 🌐 基于 HTTP 请求的稳定接口

安装依赖：

```bash
dart pub get
```
